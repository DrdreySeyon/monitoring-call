1. Les 4 modes attendus

Standard

Appel simple

Aucun TTS et aucune sequence DTMF.

TTS : - DTMF : - Usage : controle de joignabilite simple
DTMF

Touches uniquement

L'appel envoie une sequence DTMF, sans message vocal TTS.

TTS : - DTMF : #12 Delai : 4s | Intervalle : 3000ms
TTS

Voix Piper uniquement

Asterisk/Piper joue le message, sans envoyer de touches.

TTS : Bonjour, service support DTMF : - Vosk : controle la transcription
TTS + DTMF

Voix puis touches

Le TTS complet est joue, puis la sequence DTMF suit le timing.

TTS : Bonjour, besoin d'aide DTMF : 123 Delai : 4s | Intervalle : 3000ms
