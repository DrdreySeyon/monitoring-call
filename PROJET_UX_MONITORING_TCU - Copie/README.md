# PROJET_UX_MONITORING_TCU

Projet UX aligne sur la structure front/back du projet TCU.

```text
PROJET_UX_MONITORING_TCU
├── API
│   ├── main.py
│   ├── config.py
│   ├── database.py
│   ├── models.py
│   ├── ari.py
│   ├── ami_listener.py
│   ├── scheduler.py
│   ├── requirements.txt
│   └── test_backend.py
├── FRONT
│   ├── index.html
│   └── assets
│       ├── css
│       └── js
├── start_backend.bat
└── start_front.bat
```

Lancement backend + front :

```powershell
D:\bpce\PROJET_UX_MONITORING_TCU\start_backend.bat
```

Puis ouvrir :

```text
http://127.0.0.1:8090/
```
